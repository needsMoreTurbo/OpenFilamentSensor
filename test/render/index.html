Filament Flow Renderer

A lightweight player that turns simulator logs into flowing filament bars.

Quick Start

  1. Run ./pulse_simulator --log render/filament_log.csv from the test folder.
  2. Serve this directory (e.g., python3 -m http.server) and open index.html in a browser.
  3. Pick the generated CSV, then press Play to animate filament flow.
